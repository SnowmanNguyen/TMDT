Run code:
Download and Install XAMPP
Clone code
Open your XAMPP Control Panel and start the Apache and MySQL.
Extract the downloaded source code zip file.
If you are using XAMPP, copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory.
Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
Create a new database naming book_shop_db.
Import the provided SQL file. The file is known as book_shop_db.sql located inside the database folder.
Browse the Online Book Shop in a browser. i.e. http://localhost/book_shop and http://localhost/book_shop/admin for the admin side.
